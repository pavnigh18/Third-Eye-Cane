# Third Eye Cane

## Overview

Third Eye Cane is an AI-powered smart walking stick designed to assist visually impaired individuals.

## Features

- Obstacle Detection
- GPS Tracking
- Emergency SOS
- Water Detection
- Voice Alerts
- Object Recognition

## Components Used

- Arduino Uno
- NEO-6M GPS Module
- Ultrasonic Sensor
- Buzzer
- ESP32-CAM
- Battery Pack

## Future Scope

- AI Navigation
- Face Recognition
- Mobile Application Integration
